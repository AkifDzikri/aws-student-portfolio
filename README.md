# aws-student-portfolio
# AWS-Based Student Portfolio & Contact Management System

## Project Overview
This project demonstrates the deployment of a cloud-based student portfolio and contact management system using Amazon Web Services (AWS).

## AWS Services Used
- Amazon VPC
- Amazon EC2
- Amazon RDS (MySQL)
- Amazon S3

## Team Members
- Member 1: Networking (VPC)
- Member 2: Compute (EC2 & Web Server)
- Member 3: Database (RDS MySQL)
- Member 4: Storage (S3), Documentation, GitHub, Video

## Repository Purpose
This repository contains documentation and references related to the AWS-based system deployment.
